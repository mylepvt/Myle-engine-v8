iPhone Shortcut Setup (ek baar):
  1. iPhone pe ntfy app install karo (App Store — free)
  2. ntfy app open karo → "+" → topic enter karo (jo upar dala)
  3. Shortcuts app open karo → "+" → new shortcut banao
  4. Action add karo: "URL" = ntfy message ka JSON
  5. Action: "Get Contents of URL" (POST method)
  6. Action: "Create Contact" — Name: name field, Phone: phone field
  7. ntfy app me Automation set karo: "Jab notification aaye → Shortcut run karo"
